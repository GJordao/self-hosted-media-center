BookBrowser
A web-based ebook server supporting ePub and PDF. Features include search, a web-based reader, a responsive ui, and more.

Cross-Platform
BookBrowser runs on Windows, macOS, and Linux.
Easy To Use
Download it, move it to the folder with your books, run it.
Supports Multiple Formats
BookBrowser supports ePub and PDF, with more formats coming soon.
Simple and Powerful
BookBrowser has a clean UI, and also a powerful list view which allows you to see all your books at a glance.
Web-Based Reader
BookBrowser includes a highly customizable web-based reader so you can read your books from any device. You can try it here.
Sort
BookBrowser can sort books alphabetically, by series index, or by date.
Search
BookBrowser has powerful search functionality. You can search any combination of properties at once.
And more
BookBrowser has many other features as well.

System Requirements
  • BookBrowser server: Windows 7+, macOS or Linux
  • Web interface: Chrome, Firefox 3+, IE 9+, Edge, Android Browser 4.4+, Opera
  • Web-based reader: Chrome, Firefox 20+, IE 10+, Edge, Android Browser 4.4+, Opera 11+
